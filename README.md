# Blacktop

## Project Description

Blacktop is a lightweight fullscreen application designed for Android desktop setups. This project addresses a common issue found on devices like Google Pixels, where the screen remains powered on indefinitely and doesn't turn off during desktop use.

### Features
- Provides a simple fullscreen "dark mode" interface designed to enable OLED-friendly behavior.
- Built with **Svelte**, Blacktop is modern, responsive, and seamlessly buildable for GitHub Pages distribution.
- Acts as a static page solution primarily targeting browser-based runtime.

## Technical Details
Blacktop is bundled using Vite for fast builds and development.
